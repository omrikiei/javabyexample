# Java by Example

A hands-on introduction to Java using annotated example programs, inspired by [Go by Example](https://gobyexample.com/).

**Live site: [javabyexample.com](https://javabyexample.com)** (or via GitHub Pages)

## Examples

46 examples covering:

- **Basics**: Hello World, Values, Variables, Constants, For, If/Else, Switch
- **Data Structures**: Arrays, Lists, Maps
- **Functions**: Functions, Method Overloading, Varargs, Lambdas, Recursion
- **Strings**: Strings, String Formatting, Regular Expressions
- **OOP**: Classes, Records, Interfaces, Enums, Inheritance, Generics, Sealed Classes
- **Error Handling**: Exceptions
- **Concurrency**: Threads, Virtual Threads, Synchronization, Atomic Counters
- **Collections**: Sorting, Sorting by Comparator, Streams, Optional
- **I/O**: Reading Files, Writing Files, JSON
- **Time & Random**: Time, Random Numbers
- **Networking**: HTTP Client, HTTP Server
- **System**: Command-Line Arguments, Environment Variables, Logging, Processes, Exit

## Building

```bash
python3 generate.py
```

This generates the static site in the `docs/` directory.
