Java by Example: Interfaces

Interfaces are named collections of method signatures. Unlike Go, Java classes must explicitly declare that they implement an interface.

 
// A basic interface for geometric shapes.
interface Geometry {
    double area();
    double perimeter();
}

Classes implement interfaces explicitly with the implements keyword.

 
class Circle implements Geometry {
    double radius;

    Circle(double radius) {
        this.radius = radius;
    }

    public double area() {
        return Math.PI * radius * radius;
    }

    public double perimeter() {
        return 2 * Math.PI * radius;
    }
}

If a variable has an interface type, you can call methods from that interface.

 
public class Interfaces {
    static void measure(Geometry g) {
        System.out.println(g);
        System.out.println("area: " + g.area());
        System.out.println("perim: " + g.perimeter());
    }

    public static void main(String[] args) {
        var c = new Circle(5);
        measure(c);
    }
}
 
$ javac Interfaces.java
$ java Interfaces
Circle@<hash>
area: 78.53981633974483
perim: 31.41592653589793
