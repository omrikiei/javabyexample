Java by Example: Switch

switch statements express conditionals across many branches. Java’s traditional switch uses break to prevent fall-through. Since Java 14, switch expressions with arrow syntax are available.

Here’s a basic switch.

 
int i = 2;
switch (i) {
    case 1:
        System.out.println("one");
        break;
    case 2:
        System.out.println("two");
        break;
    case 3:
        System.out.println("three");
        break;
}

You can use commas to separate multiple expressions in the same case (Java 14+ arrow syntax). This also works as an expression.

 
var day = java.time.LocalDate.now().getDayOfWeek();
var type = switch (day) {
    case MONDAY, TUESDAY, WEDNESDAY,
         THURSDAY, FRIDAY -> "weekday";
    case SATURDAY, SUNDAY -> "weekend";
};  
System.out.println(type);

switch without a variable is not directly supported in Java, but you can achieve the same with if/else chains or switch expressions on booleans (Java 21+).

 
var t = java.time.LocalTime.now();
if (t.getHour() < 12) {
    System.out.println("It's before noon");
} else {
    System.out.println("It's after noon");
}
 
$ java Switch.java
two
weekday
It's before noon
