Java by Example: Random Numbers

Java’s java.util.Random and ThreadLocalRandom (or RandomGenerator in Java 17+) provide pseudo-random number generation.

 
import java.util.Random;
import java.util.concurrent.ThreadLocalRandom;

public class RandomNumbers {
    public static void main(String[] args) {
        // Random integers in [0, 100)
        var rng = new Random();
        System.out.println(rng.nextInt(100));
        System.out.println(rng.nextInt(100));

Random floats in [0.0, 1.0).

 
        System.out.printf("%.4f%n", rng.nextDouble());

Using ThreadLocalRandom for concurrent scenarios. Seeding for reproducible results.

 
        // Seeded random for reproducibility
        var seeded = new Random(42);
        System.out.println(seeded.nextInt(100));
        System.out.println(seeded.nextInt(100));
    }
}

Some of the generated numbers may differ on your machine. Seeded generators always produce the same sequence.

 
$ java RandomNumbers.java
68
41
0.6591
0
68
