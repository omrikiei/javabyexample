Java by Example: Optional

Optional (Java 8+) is a container that may or may not contain a value. It helps avoid NullPointerException and makes APIs clearer.

 
import java.util.Optional;

public class Optionals {

    // A method that may or may not return a value.
    static Optional<String> findUser(int id) {
        if (id == 1) {
            return Optional.of("Alice");
        }
        return Optional.empty();
    }

Using Optional values.

 
    public static void main(String[] args) {
        // Present value
        var user = findUser(1);
        System.out.println("present: " + user.isPresent());
        System.out.println("value: " + user.get());

        // Absent value with default
        var missing = findUser(99);
        System.out.println("absent: " + missing.orElse("unknown"));

Chaining operations on Optional.

 
        // Map and flatMap
        var upper = findUser(1)
            .map(String::toUpperCase);
        System.out.println("mapped: " + upper.orElse("N/A"));

        // ifPresent
        findUser(1).ifPresent(u ->
            System.out.println("found: " + u));
    }
}
