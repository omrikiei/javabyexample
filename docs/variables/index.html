Java by Example: Variables

In Java, variables are explicitly typed. Since Java 10, var can be used for local variable type inference.

 
String a = "initial";
System.out.println(a);

You can declare multiple variables at once.

 
int b = 1, c = 2;
System.out.println(b + " " + c);

Java will infer the type of initialized variables using var (Java 10+).

 
var d = true;
System.out.println(d);

Variables declared without initialization get default values. For instance, the default for int is 0.

 
int e;
e = 0;
System.out.println(e);

final declares a variable that cannot be reassigned after initialization.

 
final String f = "constant";
System.out.println(f);
 
$ java Variables.java
initial
1 2
true
0
constant
