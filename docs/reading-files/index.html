Java by Example: Reading Files

Reading files is a common task. Java provides several ways to read files.

 
import java.io.*;
import java.nio.file.*;

public class ReadingFiles {
    public static void main(String[] args) throws Exception {
        // Read an entire file into a string (Java 11+)
        String content = Files.readString(Path.of("/tmp/dat"));
        System.out.println(content);

Reading all lines into a list.

 
        // Read all lines
        var lines = Files.readAllLines(Path.of("/tmp/dat"));
        for (var line : lines) {
            System.out.println("line: " + line);
        }

Reading with a BufferedReader for large files.

 
        // Buffered reading
        try (var reader = Files.newBufferedReader(Path.of("/tmp/dat"))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println("buf: " + line);
            }
        }
    }
}
 
$ echo -e "hello\njava" > /tmp/dat
$ java ReadingFiles.java
hello
java
line: hello
line: java
buf: hello
buf: java
