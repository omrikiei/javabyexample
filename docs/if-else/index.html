Java by Example: If/Else

Branching with if and else in Java is straightforward.

Here’s a basic example.

 
if (7 % 2 == 0) {
    System.out.println("7 is even");
} else {
    System.out.println("7 is odd");
}

You can have an if statement without an else.

 
if (8 % 4 == 0) {
    System.out.println("8 is divisible by 4");
}

Logical operators like && and || are often useful in conditions.

 
if (8 % 2 == 0 || 7 % 2 == 0) {
    System.out.println("either 8 or 7 are even");
}

A statement can precede conditionals; any variables declared in this statement are available in the current and all subsequent branches (unlike Go, you just declare them before the if).

 
int num = 9;
if (num < 0) {
    System.out.println(num + " is negative");
} else if (num < 10) {
    System.out.println(num + " has 1 digit");
} else {
    System.out.println(num + " has multiple digits");
}

Note that you don’t need parentheses around conditions in Java… wait, actually you do! Unlike Go, Java requires parentheses around conditions.

 
$ java IfElse.java
7 is odd
8 is divisible by 4
either 8 or 7 are even
9 has 1 digit

There is no ternary if in Java’s statement form, but the ternary operator ? : works for expressions. int x = (a > b) ? a : b;
