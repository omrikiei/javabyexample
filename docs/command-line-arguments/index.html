Java by Example: Command-Line Arguments

Command-line arguments are available in the String[] args parameter of the main method.

 
public class CommandLineArgs {
    public static void main(String[] args) {
        // args contains all arguments (no program name)
        System.out.println("args: " + java.util.Arrays.toString(args));
        System.out.println("count: " + args.length);

        if (args.length >= 3) {
            System.out.println("3rd: " + args[2]);
        }
    }
}

Note that unlike Go/C, the program name is NOT included in args.

 
$ java CommandLineArgs.java a b c d
args: [a, b, c, d]
count: 4
3rd: c
