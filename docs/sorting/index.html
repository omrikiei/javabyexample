Java by Example: Sorting

Java’s Collections.sort() and Arrays.sort() provide sorting for collections and arrays.

 
import java.util.*;

public class Sorting {
    public static void main(String[] args) {
        // Sorting a list of strings
        var strs = new ArrayList<>(List.of("c", "a", "b"));
        Collections.sort(strs);
        System.out.println("Strings: " + strs);

Sorting integers.

 
        var ints = new ArrayList<>(List.of(7, 2, 4));
        Collections.sort(ints);
        System.out.println("Ints:    " + ints);

You can check if a list is already sorted.

 
        // No built-in isSorted, but easy to check
        boolean sorted = true;
        for (int i = 0; i < ints.size() - 1; i++) {
            if (ints.get(i) > ints.get(i + 1)) {
                sorted = false;
                break;
            }
        }
        System.out.println("Sorted:  " + sorted);
    }
}
 
$ java Sorting.java
Strings: [a, b, c]
Ints:    [2, 4, 7]
Sorted:  true
