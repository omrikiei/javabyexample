Java by Example: Environment Variables

Environment variables are read using System.getenv().

 
public class EnvironmentVariables {
    public static void main(String[] args) {
        // Get a specific variable
        System.out.println("HOME: " + System.getenv("HOME"));
        System.out.println("FOO:  " + System.getenv("FOO"));

List all environment variables.

 
        // All environment variables
        System.out.println();
        System.getenv().forEach((key, val) -> {
            if (key.startsWith("JAVA")) {
                System.out.println(key + "=" + val);
            }
        });
    }
}
 
$ FOO=bar java EnvironmentVariables.java
HOME: /home/user
FOO:  bar
