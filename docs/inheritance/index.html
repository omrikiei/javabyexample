Java by Example: Inheritance

Java supports class inheritance via extends. This is one of the fundamental OOP principles.

 
// Base class
class Animal {
    String name;

    Animal(String name) {
        this.name = name;
    }

    void speak() {
        System.out.println(name + " makes a sound");
    }
}

Subclasses extend the parent and can override methods using @Override.

 
class Dog extends Animal {
    Dog(String name) {
        super(name);
    }

    @Override
    void speak() {
        System.out.println(name + " barks");
    }
}

Polymorphism: a parent reference can point to a child object.

 
public class Inheritance {
    public static void main(String[] args) {
        Animal a = new Animal("Generic Animal");
        Animal d = new Dog("Rex");

        a.speak();
        d.speak();

        // instanceof check
        System.out.println("d is Dog: " + (d instanceof Dog));
    }
}
 
$ javac Inheritance.java
$ java Inheritance
Generic Animal makes a sound
Rex barks
d is Dog: true
