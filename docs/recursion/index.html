Java by Example: Recursion

Java supports recursive methods: methods that call themselves.

 
public class Recursion {

    // This fact method calls itself until it
    // reaches the base case of fact(0).
    static int fact(int n) {
        if (n == 0) {
            return 1;
        }
        return n * fact(n - 1);
    }

Unlike some languages, Java doesn’t support tail call optimization. For deep recursion, consider iterative approaches.

 
    public static void main(String[] args) {

    }
}
