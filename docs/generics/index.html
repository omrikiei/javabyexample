Java by Example: Generics

Starting with Java 5, generics let you write classes, interfaces, and methods that work with any type while providing compile-time type safety.

 
// A generic class that can hold any type.
class Box<T> {
    private T value;

    Box(T value) {
        this.value = value;
    }

    T get() {
        return value;
    }
}

Generic methods can also be defined independently.

 
import java.util.List;

public class Generics {

    // A generic method
    static <T extends Comparable<T>> T max(T a, T b) {
        return a.compareTo(b) >= 0 ? a : b;
    }

Using generic classes and methods.

 
    public static void main(String[] args) {
        var intBox = new Box<>(42);
        System.out.println("int box: " + intBox.get());

        var strBox = new Box<>("hello");
        System.out.println("str box: " + strBox.get());

        System.out.println("max(3,5): " + max(3, 5));
        System.out.println("max(a,z): " + max("a", "z"));
    }
}
 
$ javac Generics.java
$ java Generics
int box: 42
str box: hello
max(3,5): 5
max(a,z): z
