Java by Example: Lambdas

Java supports lambda expressions (since Java 8), which are anonymous functions. They are commonly used with functional interfaces.

 
import java.util.function.IntBinaryOperator;
import java.util.function.IntUnaryOperator;

public class Lambdas {

This method takes a function (functional interface) as a parameter.

 
    static int apply(IntBinaryOperator op, int a, int b) {
        return op.applyAsInt(a, b);
    }

Lambdas can capture variables from their enclosing scope (effectively final variables).

 
    public static void main(String[] args) {
        // A simple lambda
        IntBinaryOperator add = (a, b) -> a + b;
        System.out.println(apply(add, 1, 2));

You can also use lambdas inline.

 
        System.out.println(apply((a, b) -> a * b, 3, 4));

Lambdas can be used to create closures. This function returns a lambda that adds x to its argument.

 
        int x = 5;
        IntUnaryOperator addX = n -> n + x;
        System.out.println(addX.applyAsInt(10));
    }
}
