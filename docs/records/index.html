Java by Example: Records

Records (Java 16+) are a concise way to create immutable data classes. They automatically generate equals(), hashCode(), toString(), and accessor methods.

 
// A record automatically creates a constructor,
// accessors, equals, hashCode, and toString.
record Point(int x, int y) {}

Records can have custom methods.

 
record Rect(double width, double height) {
    double area() {
        return width * height;
    }
}

Using records.

 
public class Records {
    public static void main(String[] args) {
        var p = new Point(3, 4);
        System.out.println(p);
        System.out.println("x: " + p.x());

        var r = new Rect(3, 4);
        System.out.println("area: " + r.area());

        // Records implement equals based on values
        var p2 = new Point(3, 4);
        System.out.println("equal: " + p.equals(p2));
    }
}
 
$ javac Records.java
$ java Records
Point[x=3, y=4]
x: 3
area: 12.0
equal: true
