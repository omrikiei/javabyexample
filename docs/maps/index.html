Java by Example: Maps

Maps (via HashMap) are Java’s built-in associative data type (sometimes called hashes or dicts in other languages).

 
import java.util.HashMap;
import java.util.Map;

public class Maps {
    public static void main(String[] args) {

To create an empty map, use new HashMap<>().

 
        var m = new HashMap<String, Integer>();

Set key/value pairs using put().

 
        m.put("k1", 7);
        m.put("k2", 13);
        System.out.println("map: " + m);

Get a value for a key with get().

 
        System.out.println("v1: " + m.get("k1"));

The size() method returns the number of key/value pairs.

 
        System.out.println("len: " + m.size());

The remove() method deletes key/value pairs.

 
        m.remove("k2");
        System.out.println("map: " + m);

containsKey() checks for the presence of a key.

 
        System.out.println("has: " + m.containsKey("k2"));

Map.of() creates an immutable map inline (Java 9+).

 
        var n = Map.of("foo", 1, "bar", 2);
        System.out.println("map: " + n);
    }
}
