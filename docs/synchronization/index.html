Java by Example: Synchronization

Java provides several mechanisms for synchronizing access to shared data across threads.

 
import java.util.concurrent.locks.ReentrantLock;

public class Synchronization {
    static int counter = 0;
    static final Object lock = new Object();

The synchronized keyword is the simplest way to achieve mutual exclusion.

 
    static synchronized void incrementSync() {
        counter++;
    }

You can also synchronize on specific objects or use ReentrantLock for more control.

 
    static final ReentrantLock rLock = new ReentrantLock();

    static void incrementWithLock() {
        rLock.lock();
        try {
            counter++;
        } finally {
            rLock.unlock();
        }
    }

Example usage with multiple threads.

 
    public static void main(String[] args) throws Exception {
        counter = 0;
        var threads = new Thread[1000];
        for (int i = 0; i < threads.length; i++) {
            threads[i] = new Thread(() -> {
                for (int j = 0; j < 1000; j++) {
                    incrementSync();
                }
            });
            threads[i].start();
        }
        for (var t : threads) t.join();
        System.out.println("count: " + counter);
    }
}

The count will always be exactly 1,000,000 thanks to synchronization.

 
$ java Synchronization.java
count: 1000000
