Java by Example: Strings

Java String is an immutable sequence of characters. The standard library provides a rich set of string methods.

 
public class Strings {
    public static void main(String[] args) {
        // String literals
        var s = "Hello, World";
        System.out.println(s);

Common string operations.

 
        System.out.println("Contains:  " + s.contains("World"));
        System.out.println("Count:     " + s.chars().filter(c -> c == 'l').count());
        System.out.println("StartsWith:" + s.startsWith("Hello"));
        System.out.println("EndsWith:  " + s.endsWith("World"));
        System.out.println("IndexOf:   " + s.indexOf("World"));
        System.out.println("Length:    " + s.length());

String transformation methods return new strings since Strings are immutable.

 
        System.out.println("ToUpper:   " + s.toUpperCase());
        System.out.println("ToLower:   " + s.toLowerCase());
        System.out.println("Replace:   " + s.replace("World", "Java"));
        System.out.println("Substring: " + s.substring(0, 5));
        System.out.println("Trim:      " + "  hi  ".trim());

Text blocks (Java 13+) for multi-line strings.

 
        var tb = """
                Hello,
                Java!
                """;
        System.out.println(tb);
    }
}
