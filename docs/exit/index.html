Java by Example: Exit

Use System.exit() to immediately exit with a given status code.

 
public class Exit {
    public static void main(String[] args) {
        // This will be printed
        System.out.println("starting");

        // Exit with status 3
        // Note: finally blocks and shutdown hooks
        // may or may not run depending on the JVM
        System.exit(3);

        // This will never be reached
        System.out.println("unreachable");
    }
}

If you run exit, it will exit with status 3. The shell will show the exit code.

 
$ java Exit.java
starting
$ echo $?
3

Note that unlike Go, the main method returning normally exits with status 0. Use System.exit(code) for non-zero exits.
