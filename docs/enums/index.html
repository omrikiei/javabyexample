Java by Example: Enums

Enum types in Java are full classes that represent a fixed set of constants. They can have fields, methods, and constructors.

 
// A basic enum.
enum Direction {
    NORTH, SOUTH, EAST, WEST
}

Enums can have fields and methods.

 
enum Planet {
    MERCURY(3.303e+23, 2.4397e6),
    VENUS(4.869e+24, 6.0518e6),
    EARTH(5.976e+24, 6.37814e6);

    final double mass;
    final double radius;

    Planet(double mass, double radius) {
        this.mass = mass;
        this.radius = radius;
    }

    double surfaceGravity() {
        final double G = 6.67300E-11;
        return G * mass / (radius * radius);
    }
}

Using enums.

 
public class Enums {
    public static void main(String[] args) {
        var d = Direction.NORTH;
        System.out.println(d);

        // Enums work with switch
        switch (d) {
            case NORTH -> System.out.println("Going up!");
            case SOUTH -> System.out.println("Going down!");
            default -> System.out.println("Going sideways!");
        }

        // Enum methods
        System.out.printf("Earth gravity: %.2f m/s²%n",
            Planet.EARTH.surfaceGravity());

        // Iterate over all values
        for (var p : Planet.values()) {
            System.out.println(p.name());
        }
    }
}
 
$ javac Enums.java
$ java Enums

Going up!
Earth gravity: 9.80 m/s²
