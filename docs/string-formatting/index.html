Java by Example: String Formatting

Java offers several ways to format strings. String.format, printf, and the newer formatted() method (Java 15+).

 
public class StringFormatting {

    record Point(int x, int y) {}

    public static void main(String[] args) {
        var p = new Point(1, 2);

General formatting with printf / format.

 
        // Print a representation of the object
        System.out.printf("struct1: %s%n", p);

        // String formatting
        System.out.printf("string: %s%n", "hello");

        // Integer formatting
        System.out.printf("int:     %d%n", 42);

Float formatting and width control.

 
        System.out.printf("float:   %.2f%n", 3.14159);
        System.out.printf("float:   %e%n", 100000.0);

        // Right-justify in a 10-char wide field
        System.out.printf("right:   |%10d|%n", 42);
        // Left-justify
        System.out.printf("left:    |%-10d|%n", 42);

String.format returns a string without printing it.

 
        String s = String.format("format: %s has %d points", "Java", 100);
        System.out.println(s);
    }
}
