Java by Example: JSON

Java doesn’t have built-in JSON support in the standard library, but common libraries like Jackson and Gson are widely used. Here we show a simple approach using the built-in javax.json or manual parsing concepts.

 
import java.util.*;

// For simple cases, you can represent JSON
// with Maps and Lists. For real projects,
// use Jackson or Gson.
public class Json {
    public static void main(String[] args) {
        // Building a JSON-like structure
        var person = new LinkedHashMap<String, Object>();
        person.put("name", "Alice");
        person.put("age", 30);

Nested structures and arrays.

 
        person.put("hobbies", List.of("reading", "coding"));

        // Simple serialization (not real JSON,
        // but demonstrates the concept)
        System.out.println(person);

For proper JSON, use a library like Jackson:

 
        // With Jackson (pseudo-code):
        // ObjectMapper mapper = new ObjectMapper();
        // String json = mapper.writeValueAsString(person);
        // Person p = mapper.readValue(json, Person.class);
        System.out.println("name: " + person.get("name"));
    }
}

For production use, add Jackson or Gson to your project dependencies.

 
$ java Json.java
{name=Alice, age=30, hobbies=[reading, coding]}
name: Alice
