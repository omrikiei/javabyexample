Java by Example: Method Overloading

Java supports method overloading: multiple methods with the same name but different parameter lists. This is a form of compile-time polymorphism.

Here we define two add methods — one for int and one for double.

 
public class MethodOverloading {

    static int add(int a, int b) {
        return a + b;
    }

    static double add(double a, double b) {
        return a + b;
    }

You can also overload on the number of parameters.

 
    static int add(int a, int b, int c) {
        return a + b + c;
    }

Java selects the correct method based on the argument types at compile time.

 
    public static void main(String[] args) {
        System.out.println(add(1, 2));
        System.out.println(add(1.5, 2.5));
        System.out.println(add(1, 2, 3));
    }
}
 
$ java MethodOverloading.java
3
4.0
