Java by Example: Constants

Java supports constants using the final keyword. final can be used with fields, local variables, and method parameters.

 
public class Constants {
    // Class-level constants are typically
    // declared as static final.
    static final String S = "constant";
    static final int N = 500000000;

A constant expression can be used anywhere a value of its type is expected.

 
    public static void main(String[] args) {
        System.out.println(S);

Constant expressions perform arithmetic with arbitrary precision at compile time.

 
        double d = 3e20 / N;
        System.out.println(d);

A numeric constant has no type until it’s given one, such as by a cast or assignment.

 
        System.out.println((long) d);
    }
}
 
$ java Constants.java
constant
6.0E11
600000000000
