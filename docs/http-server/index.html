Java by Example: HTTP Server

Java includes a simple built-in HTTP server via com.sun.net.httpserver. For production, use frameworks like Spring Boot or Javalin.

 
import com.sun.net.httpserver.*;
import java.io.*;
import java.net.InetSocketAddress;

public class HttpServer {

    static void hello(HttpExchange exchange) throws IOException {
        var response = "hello\n";
        exchange.sendResponseHeaders(200, response.length());
        try (var os = exchange.getResponseBody()) {
            os.write(response.getBytes());
        }
    }

Setting up request handlers.

 
    static void headers(HttpExchange exchange) throws IOException {
        var headers = exchange.getRequestHeaders();
        var sb = new StringBuilder();
        headers.forEach((key, values) ->
            sb.append(key).append(": ")
              .append(String.join(", ", values)).append("\n"));
        var response = sb.toString();
        exchange.sendResponseHeaders(200, response.length());
        try (var os = exchange.getResponseBody()) {
            os.write(response.getBytes());
        }
    }

Starting the server.

 
    public static void main(String[] args) throws Exception {
        var server = com.sun.net.httpserver.HttpServer
            .create(new InetSocketAddress(8090), 0);
        server.createContext("/hello", HttpServer::hello);
        server.createContext("/headers", HttpServer::headers);
        server.start();
        System.out.println("Server on :8090");
    }
}

Run the server in the background and test with curl.

 
$ java HttpServer.java &
$ curl localhost:8090/hello
hello
