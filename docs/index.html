Java by Example

Java is a widely-used, object-oriented programming language designed for building robust, secure, and high-performance applications. Please read the official documentation to learn more.

Java by Example is a hands-on introduction to Java using annotated example programs. Check out the first example or browse the full list below.

Unless stated otherwise, examples here assume Java 21+ and may use newer language features like records, sealed classes, pattern matching, and virtual threads.
