Java by Example: Threads

Java has built-in support for concurrent execution via threads.

 
public class Threads {

    static void task(String name) {
        for (int i = 0; i < 3; i++) {
            System.out.println(name + ": " + i);
        }
    }

Start a new thread using Thread.start() or Thread.ofVirtual() (Java 21+).

 
    public static void main(String[] args) throws Exception {
        // Traditional thread
        var t1 = new Thread(() -> task("thread1"));
        t1.start();

        // Virtual thread (Java 21+)
        var t2 = Thread.ofVirtual().start(() -> task("thread2"));

        // Run in the main thread too
        task("main");

        // Wait for threads to finish
        t1.join();
        t2.join();
        System.out.println("done");
    }
}

When we run this program, we see the output of the threads interleaved. The exact ordering may vary between runs.

 
$ java Threads.java
main: 0
main: 1
main: 2
thread1: 0
thread1: 1
thread1: 2
thread2: 0
thread2: 1
thread2: 2
done
