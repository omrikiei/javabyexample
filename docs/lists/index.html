Java by Example: Lists

Lists in Java (via ArrayList) are resizable sequences, similar to Go’s slices. They are one of the most commonly used data structures.

 
import java.util.ArrayList;
import java.util.List;

public class Lists {
    public static void main(String[] args) {

To create an empty list of strings, use new ArrayList<>().

 
        var s = new ArrayList<String>();
        System.out.println("emp: " + s);

We can add elements to the list.

 
        s.add("a");
        s.add("b");
        s.add("c");
        System.out.println("add: " + s);

Individual elements can be accessed with get. The size() method gives the list length.

 
        System.out.println("get: " + s.get(2));
        System.out.println("len: " + s.size());

Lists support many operations like subList (similar to Go’s slicing).

 
        System.out.println("sub: " + s.subList(1, 3));

List.of() creates an immutable list inline (Java 9+).

 
        var t = List.of("a", "b", "c");
        System.out.println("dcl: " + t);
    }
}
 
$ java Lists.java
emp: []
add: [a, b, c]
get: c
len: 3
sub: [b, c]
dcl: [a, b, c]
