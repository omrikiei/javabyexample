Java by Example: Varargs

Varargs (variable-length arguments) let methods accept any number of arguments of a given type.

 
public class Varargs {

    // This method takes an arbitrary number
    // of ints.
    static int sum(int... nums) {
        int total = 0;
        for (int n : nums) {
            total += n;
        }
        return total;
    }

Varargs methods can be called with individual arguments or an array.

 
    public static void main(String[] args) {
        System.out.println(sum(1, 2));
        System.out.println(sum(1, 2, 3));

        int[] nums = {1, 2, 3, 4};
        System.out.println(sum(nums));
    }
}
