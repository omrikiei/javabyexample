Java by Example: Arrays

In Java, an array is a fixed-size sequence of elements of a specific type.

 
// Create an array that holds exactly 5 ints.
// By default, elements are zero-valued.
int[] a = new int[5];
System.out.println("emp: " + java.util.Arrays.toString(a));

We can set a value at an index and get a value with the standard array[index] syntax.

 
a[4] = 100;
System.out.println("set: " + java.util.Arrays.toString(a));
System.out.println("get: " + a[4]);

The built-in length field returns the length of an array.

 
System.out.println("len: " + a.length);

Use this syntax to declare and initialize an array in one line.

 
int[] b = {1, 2, 3, 4, 5};
System.out.println("dcl: " + java.util.Arrays.toString(b));

Multi-dimensional arrays are also supported.

 
int[][] twoD = new int[2][3];
for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 3; j++) {
        twoD[i][j] = i + j;
    }
}
System.out.println("2d: " + java.util.Arrays.deepToString(twoD));

Note that arrays are printed in [a, b, c] form when using Arrays.toString.

 
$ java Arrays.java
emp: [0, 0, 0, 0, 0]
set: [0, 0, 0, 0, 100]
get: 100
len: 5
dcl: [1, 2, 3, 4, 5]
2d: [[0, 1, 2], [1, 2, 3]]
